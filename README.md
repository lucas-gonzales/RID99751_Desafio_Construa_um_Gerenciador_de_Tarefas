# The Dev News

Blog responsivo desenvolvido para o desafio de CSS da DNC.

## Sobre

Projeto de um blog fictício chamado "The Dev News" com foco em responsividade. O layout se adapta de celulares (320px) até monitores ultrawide.

## Tecnologias

- HTML5
- CSS3
- Google Fonts (Montserrat)

## Como usar

Clone o repositório e abra o `index.html` no navegador:

```bash
git clone https://github.com/lucas-gonzales/RID99751_Desafio-Desenvolvendo-um-Blog-Responsivo.git
cd RID99751_Desafio-Desenvolvendo-um-Blog-Responsivo
```

## Estrutura

```
├── index.html
├── style.css
└── README.md
```

## Layout

O design foi baseado no protótipo do [Figma](https://www.figma.com/design/tI1YsslgD5Oao1tAclxlfA/Blog-responsivo---DNC).

**Desktop:** Banner + sidebar lado a lado, posts em grid de 2 colunas

**Mobile:** Layout em coluna única, elementos empilhados
